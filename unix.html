Useful Unix Tips

Commands and More

df - disk space free
My favorite is 'df -h' which reports disk usage and free space for all mounted volumes in the highest sensible units (K, M, G, etc., "human-readable").
du - disk usage
My favorites are 'du -hs <directory>' and 'du -hs *' which report the total disk usage of a directory or all files and directories in the current directory, respectively. The '-h' option makes the output "human-readable".
echo
Echoes its arguments to standard output. The -e switch (under bash) lets you use things like \n and \t in the text and have them turn into newlines and tabs. Under sh, \n and \t ae interpreted automatically.
find
Reports the full pathname of files matching certain criteria, and optionally executes a command on them. I use "find <directory> -iname '*.mp3'" to list all MP3 files found below the given directory. The single quotes are needed to prevent the shell from expanding (globbing?) the *. The -iname option says to look for matching names in a case-insensitive manner.
~/.gvfs
The directory where Ubuntu mounts remote drives that you connect to manually, through the "Connect to Server..." dialog.
ls - list files
My favorite variant is 'ls -oh', which provides permissions, owner (but not group), size in highest sensible units (K, M, G, etc.), modification time, and filename.
which
Specifies where a command is found without executing it. Useful for detecting which software you have installed.
